\begin{itemize}
\item $\ds \int_0^3 s(t)\ dt = 10,$
\item $\ds \int_3^5 s(t) \ dt=8,$
\item $\ds \int_3^5 r(t)\ dt = -1,$ and
\item $\ds \int_0^5 r(t)\ dt = 11.$
\end{itemize}
Use these values to evaluate the given definite integrals.}{
